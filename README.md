# VirtualWorks Task 2: Creative Profile Studio

This version turns the POST-request task into a small profile-card studio.

Users enter:

- Name
- Role
- Bio
- Image URL
- Vibe
- Palette
- Skills

The frontend sends the details to the backend with `POST /api/profile`. The backend validates the request, creates initials, formats skills, selects the requested palette, and returns both structured profile data and ready-to-render card HTML.

## Run

```bash
npm start
```

Open:

```text
http://localhost:5012
```

If port `5012` is already busy:

```bash
$env:PORT=5013; npm start
```

## API

### `POST /api/profile`

Request body:

```json
{
  "name": "Rajvee Sharma",
  "role": "Frontend Developer",
  "bio": "I create clean, practical web interfaces and enjoy connecting frontend details with backend logic.",
  "imageUrl": "https://example.com/photo.jpg",
  "vibe": "Focused",
  "palette": "aurora",
  "skills": "HTML, CSS, JavaScript, Node.js"
}
```

Success response:

```json
{
  "success": true,
  "message": "Profile studio card generated by the backend.",
  "profile": {
    "name": "Rajvee Sharma",
    "role": "Frontend Developer",
    "skills": ["HTML", "CSS", "JavaScript", "Node.js"],
    "initials": "RS",
    "tagline": "Focused Frontend Developer"
  },
  "cardHtml": "<article class=\"profile-card\">...</article>"
}
```
